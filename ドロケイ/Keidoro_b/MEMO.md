# ドロケイ



# 初期設定
ゲーム開始前に以下の三つの座標を設定してください  
- 初期座標 ... 開始、終了時のTP座標  
- 復活座標 ... 泥棒の復活座標  
- 牢屋座標 ... 泥棒の牢屋座標  

<h2 id="init_setting">設定方法</h2>

`/function keidoro/book` で進行ブックを入手できます  
開いて **座標設定** から各座標の設定をできます



# ゲーム進行
## 1. 警察の設定
ゲーム開始前に **参加状態** のプレイヤーを **警察** に設定しておく必要があります。  
警察の設定は **進行ブック** から行えます。  


## 2. 参加
プレイヤーを **参加状態** にします。  
`詳細`: タグ(`kd_join`)を付与、初期位置にTP    
`コマンド`: `/function keidoro/join` (プレイヤー自身が実行)  


## 3. 開始
ゲームの開始は **進行ブック** から行えます。  
> 開始には **警察**、**泥棒** が各1人以上参加している必要があります。  

`泥棒`: タグ(`kd_join`)を削除、タグ(`kd_play, kd_thief`)を付与、初期位置にTP  
`警察`: タグ(`kd_join`)を削除、タグ(`kd_play, kd_police`)を付与、牢屋位置にTP  


## 4. 警察開放
ゲーム開始から **30秒後** に警察が牢屋から解放されます。  
開放時間の設定は **進行ブック** から行えます。  


## 5. 終了
**残り時間が0秒になる** or **泥棒が全員捕まる** のどちらかの条件をみたすとゲームが終了します。

`詳細`: リザルトを表示、タグ(`kd_join`)を付与、タグ(`kd_play, kd_thief, kd_police`)を削除、初期位置にTP  

# 捕まった際の処理
警察は泥棒を殴ると捕まえることができます。

捕まった際に 残機が
- 残機が 0  
  牢屋にテレポートされます。  

- 残機が 1以上
  残機が一つ減り、復活位置にテレポートします。  
  無敵時間が30秒付与され、透明、ダメージ無効となります。



発電機について
- 場所はマーカー(赤)で把握できる
- 半径2m以内で レバーをカチカチすることで電力をためれる
- 電力が最大まで貯まると泥棒全員にグレネードが配られる
- グレネード配布語は電力がリセットされ、クールタイムが発生
- 発電機は最大3人まで同時発電が可能


# 最終確認用
